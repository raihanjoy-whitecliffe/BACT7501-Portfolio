# Week 3 - While Loops and Lists

## Learning Outcome

Through this activity I have learned how to use while loops to keep repeating the logic until a specific condition is met. I have also learned how to store multiple parts of data inside lists and how to add, remove, or search through those items.

## Activity Description

I have completed a few tasks focused on loops and lists. I've built a grade calculator which forces the user to type a valid score between 0 and 100 and a number guessing game that gives "too high" or "too low" hints. I have also created programs to count how many grades marked as passed from a list of scores, removed negative numbers, and build a shopping list which lets you to add items and also to delete the last one you entered.

## Files Included

* task1_gradingsystem.py


* task2_game.py


* task3_scores.py


* task4_shopping.py


* task5_rewrite.py



## Skills Demonstrated

* Using while loops to block users until they enter valid data or guess correctly


* Modifying lists dynamically using .append() and del


* Using for loops to filter and count specific items inside a list


* Accessing list items using index numbers, including negative indexing for the last item



## Reflection

The hardest part was making sure that my while loops actually stopped running. I've realized that if I didn't ask the user for a new input inside the loop, the program would just get stuck inside an infinite loop. 
I have also practiced removing, and adding data in the list to make sure I was deleting the correct item from my shopping list.
After completing these tasks I have understood how important lists are for multiple data, rather than making separate variables.
