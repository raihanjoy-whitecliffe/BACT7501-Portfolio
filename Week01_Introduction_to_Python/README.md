Week 1 - Python Basics and Core Fundamentals



Learning Outcome

During this activity, I have learned how to get inputs from the user and worked with different data types like string, integer, and floats.
I have also learned how to execute basic math operations, fixing decimals to 2 d.p using f-strings and also used If else conditional statements.



Activity Description

I have completed a series of basic python tasks which are designed to practice how variable, math calculations are done. 
The task included building a  profile generator by tracking age, converting height, a grade calculator which marks assignments and exams, 
a budget tool to track balance, a simple billing process and printing receipt which we see in stores with automatic gst calculations, another program to check the eligibility of getting a scholarship



Files Included

- profile_summary.py
- grade_calculator.py  
- budget_calculator.py
- billing_system.py
- scholarship_checker.py

Skills Demonstrated

Capturing user input and type casting (int(), float(), str())  
Performing arithmetic conversions and percentage weight calculations  
Showing clean terminal outputs using f-strings with deicmal place limit (:.2f)  
Implementing basic conditional logic and nested if structures to evaluate multiple criteria  

Reflection

The most challenging part of this activity was keeping track of data types during math operations and making sure the final print statements looked clean. It is easy to forget that default input type is string, 
so making sure prices were converted to floats and quantities to integers was crucial to avoid crashes. Finding out the syntax for rounding floats to two decimal places in f-strings also needed practice to give a proper look on the final receipts.
From completing this, I learned exactly how strict Python is with data types and how important  a Clean UI experience is, for the end user. By completing these simple tasks I've understood how real world programs handle data in the back-end before showing the results to the users
